# 爱关怀企业创建接口



## 创建企业

***GET*** /v1/corp/create

创建企业

*GET参数*


参数 | 必须 | 描述
----|------|----
abbrName|是|企业简称
fullName|是|企业全称
contactName|是|联系人
tel |是|联系人手机
email|是|邮箱
feature|否|点滴关怀版本,默认1,1=融合版，2=爱关怀

返回：

```
{
	"errno": 0,
	"error": "error info",
	"body": {
		"corpId": 123
	}
}
```


errno=0 表示成功。

corpId: 企业ID
